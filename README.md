# Dotfiles

Only tested on macOS

- `brew install --cask mediahuman-audio-converter appcleaner onyx kid3 font-recursive-mono-nerd-font ghostty flowvision hammerspoon aria2d squirrel-app miniconda keka macpacker iina pearcleaner ocenaudio`
- `brew install neovim fzf eza yazi zoxide aria2 ripgrep coreutils grep gawk gnu-sed ffmpegthumbnailer unar jq poppler fd xclip ruff mdformat mpv mihomo proxychains-ng clippy lua-language stylua media-info p7zip mdfried biome npm markdown-oxide imagemagick gitui gitu alt-tab`
- `brew install acsandmann/tap/rift`

## Services

```sh
rift service start
```
