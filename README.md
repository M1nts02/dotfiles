# Dotfiles

Mainly used on macOS

## Install

- `brew install --cask aerospace font-recursive-mono-nerd-font sf-symbols wezterm`
- `brew install neovim gitui fzf mpv helix eza yazi zoxide pueue aria2 asdf bottom musikcube ripgrep starship`
- `brew install FelixKratz/formulae/sketchybar`
  - `brew services start sketchybar`
